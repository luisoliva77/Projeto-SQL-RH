# Dashboard Gestão de Pessoas

## 📌 Sobre o Dashboard

Este dashboard foi desenvolvido no Power BI com o objetivo de analisar indicadores estratégicos relacionados à gestão de pessoas dentro de uma organização.

Através dos dados processados e estruturados no SQL Server, foi possível criar visualizações analíticas que auxiliam na interpretação de informações importantes sobre funcionários, departamentos, salários, benefícios e faltas.

---

## 📈 Indicadores Apresentados

O dashboard permite visualizar informações como:

* Total de funcionários ativos
* Taxa de turnover
* Distribuição de funcionários por gênero
* Quantidade de faltas por motivo
* Funcionários por cidade
* Estrutura hierárquica por departamento e cargo
* Indicadores salariais
* Benefícios por funcionário

---

## 🎯 Objetivo

O principal objetivo do dashboard é transformar dados em informações visuais e estratégicas, facilitando a análise de indicadores de RH e apoiando a tomada de decisão.

---

## 🛠 Ferramenta Utilizada

* 📊 Power BI

## 📌 Insights Possíveis

Com o dashboard é possível identificar:

* Departamentos com maior quantidade de funcionários
* Distribuição entre homens e mulheres
* Principais motivos de faltas
* Funcionários com maiores benefícios
* Indicadores gerais de gestão de pessoas

---

## 🚀 Resultado

O projeto permitiu praticar conceitos de visualização de dados, construção de dashboards interativos e análise de indicadores organizacionais utilizando Power BI.
